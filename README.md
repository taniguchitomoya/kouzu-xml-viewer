# 法務局 地図XML（公図）ビューア

> [!NOTE]
> 本プロジェクトのソースコードおよびドキュメントは、AI（Google Gemini）によって生成されています。

法務局から提供される「地図XML」データ（公図データ）をブラウザ上で手軽に閲覧するための静的ウェブアプリケーションです。
サーバーサイドは完全に静的ファイルのみで構成されており、ブラウザ（JavaScript）上でXMLの解析および描画を行います。

## ✨ 主な機能

- **ZIP/XMLファイルの直接読み込み**
  - G空間情報センター等からダウンロードした地図XMLのZIPファイル（または展開したXMLファイル）を、ドラッグ＆ドロップで直接読み込み可能です。
  - `js/jszip.min.js` を使用し、クライアントサイドで高速にZIPの解凍とスキャンを行います。
- **公図（筆）の可視化**
  - 公図の境界線や筆（土地）の形状を描画します。
  - 丁目ごとに自動で色分けされ、視覚的に分かりやすくなっています。
- **インタラクティブな操作**
  - マウスホイールやボタンによる拡大・縮小、ドラッグによる移動（パン操作）に対応。
  - 筆をクリックまたはホバーすると、地番や詳細情報（座標系、市区町村、縮尺など）を表示します。
- **地番検索機能**
  - 特定の地番を入力して、該当する筆を素早く検索・ハイライトできます。
- **印刷用モード**
  - 背景を白基調にした、印刷やキャプチャに適したスタイルに切り替えることができます。
- **ローカルデータの自動検出**
  - `data/` フォルダ配下に置かれたファイルを自動検出し、画面上からクリックするだけで読み込めます。

## 🚀 クイックスタート

### 動作環境
- Python 3 がインストールされている環境（WSL/Linux, macOS, Windowsなど）
- 最新のモダンブラウザ（Chrome, Edge, Firefox, Safariなど）

### 起動手順
1. リポジトリのルートディレクトリで以下のコマンドを実行します。
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
2. 自動的にブラウザが立ち上がり、 `http://localhost:8000/` が開きます。
3. もし自動で開かない場合は、ブラウザで直接 `http://localhost:8000/` にアクセスしてください。

## 🗺️ 地図データの入手について

本ビューアで閲覧する地図データは、**G空間情報センター**などからダウンロードした法務局地図XMLデータ（任意座標系、公共座標系ともに対応）をご使用いただけます。
ダウンロードしたZIPファイルのまま、ブラウザ上のドロップゾーンへドラッグ＆ドロップしてご活用ください。
