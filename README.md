# 法務局 地図XML（公図）ビューア

> [!NOTE]
> 本プロジェクトのソースコードおよびドキュメントは、AI（Google Gemini）によって生成されています。

法務局から提供される「地図XML」データ（公図データ）をブラウザ上で手軽に閲覧するための静的ウェブアプリケーションです。
サーバーサイドは完全に静的ファイルのみで構成されており、ブラウザ（JavaScript）上でXMLの解析および描画を行います。

## ✨ 主な機能

- **ZIP/XMLファイルの直接読み込み**
  - G空間情報センター等からダウンロードした地図XMLのZIPファイル（または展開したXMLファイル）を、ドラッグ＆ドロップで直接読み込み可能です。
  - `js/zip.min.js` を使用し、HTTP Range Requestsを用いた効率的な部分読み込みやクライアントサイドでのZIP解凍・スキャンを行います。
- **公図（筆）の可視化**
  - 公図の境界線や筆（土地）の形状を描画します。
  - 丁目ごとに自動で色分けされ、視覚的に分かりやすくなっています。
- **インタラクティブな操作**
  - マウスホイールやボタンによる拡大・縮小、ドラッグによる移動（パン操作）に対応。
  - 筆をクリックまたはホバーすると、地番や詳細情報（座標系、市区町村、縮尺など）を表示します。
- **地番検索機能**
  - 特定の地番を入力して、該当する筆を素早く検索・ハイライトできます。
- **印刷用モード**
  - 背景を白基調にした、印刷やキャプチャに適したスタイルに切り替えることができます。
- **ローカルデータの自動検出**
  - `public/data/` フォルダ配下に置かれたファイルを自動検出し、画面上からクリックするだけで読み込めます。

- Python 3 がインストールされている環境（WSL/Linux, macOS, Windowsなど）
- 最新のモダンブラウザ（Chrome, Edge, Firefox, Safariなど）

### 起動手順
1. リポジトリのルートディレクトリで以下のコマンドを実行します。
   ```bash
   chmod +x start.sh
   ./start.sh
   ```
2. 自動的にブラウザが立ち上がり、 `http://localhost:8000/` が開きます。
3. もし自動で開かない場合は、ブラウザで直接 `http://localhost:8000/` にアクセスしてください。

## 🏎️ 高速化とキャッシュ機構

本ビューアは、多数の地図XMLが含まれる巨大なZIPファイルを高速かつ軽量に読み込むため、以下の仕組みを導入しています。

- **HTTP Range Requests（範囲要求）による部分読み込み**
  - リモートサーバー（`data/` ディレクトリ配下など）のZIPファイルを読み込む際、ZIPファイル全体をダウンロードするのではなく、`zip.js` の `HttpReader` を用いて、ZIP末尾のインデックス情報（セントラルディレクトリ）のみをピンポイントで取得します。
  - プルダウンで特定の地図（丁目）を選択したタイミングで初めて、該当するXMLファイルのデータのみをサーバーから部分取得して展開します。
- **地域名（地図名）のローカルキャッシュ (`localStorage`)**
  - ZIPファイル内の各XMLデータに記述されている「地図名（地域名）」のリストを、初回ロード時にパースしてブラウザの `localStorage` にキャッシュします。
  - 2回目以降の読み込み時は、ネットワーク通信や解凍処理を一切発生させず、キャッシュから瞬時に図幅リストを再構築します。
  - キャッシュの識別キーは、リモートファイルの場合は `url:<URL>`、ローカルファイルの場合は `file:<ファイル名>:<サイズ>:<更新日時>` となります。
- **容量上限時の自動クリーンアップ**
  - ブラウザの `localStorage` 容量上限（約5MB）に達して `QuotaExceededError` が発生した場合は、本アプリが作成した過去の地域名キャッシュデータを自動で全クリアして空き容量を確保し、処理を安全に続行します（地図表示自体に影響はありません）。

## 🔒 セキュリティポリシー

- **完全ローカル完結**
  - 読み込んだZIP/XMLデータはすべてクライアント（ブラウザ）のメモリ上で処理され、外部のサーバーへ送信・保存されることはありません。
- **外部依存の排除**
  - 外部のCDNやライブラリのロードは行わず、静的セキュリティ監査済みのコードのみ（[zip.min.js](file:///home/tomoya/kouzu/public/js/zip.min.js) 等）で自立動作します。

## 🗺️ 地図データの入手について

本ビューアで閲覧する地図データは、**G空間情報センター**などからダウンロードした法務局地図XMLデータ（任意座標系、公共座標系ともに対応）をご使用いただけます。
ダウンロードしたZIPファイルのまま、ブラウザ上のドロップゾーンへドラッグ＆ドロップしてご活用ください。
